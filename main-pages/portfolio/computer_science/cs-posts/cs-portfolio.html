Autonomous Raspberry Pi Rover | C++ | Python | HTML

This is a partially autonomous rover that was designed collaboratively between me and two of my classmates for a Computer Science course. The rover hosts a web interface via the Raspberry Pi on the back that allows for wireless integration on a computer. The web interface was constructed with basic HTML and has a text box where you input commands. While it had the ability to go forward, turn around, and stop the main feature was that It could navigate to geographic coordinates.

The way this works is that the Raspberry Pi on the back of the rover hosts a server using flask with the basic web interface mentioned before. The server is hosted on local WiFi but once the instructions are given the rover autonomously navigates to the coordinates provided with or without lasting connection to the network. The instructions are taken from the Raspberry Pi and given to the Arduino via a serial port and the Arduino interprets those instructions and sends signals to the hardware to move.

Consumer Producer Problem w/ bounded buffer | Java

This is my attempt at solving the producer consumer problem using bounded buffer in Java. There are two threads, one consumer that removes items from the buffer and 'consumes' them, and one producer that adds items to the buffer, thus 'producing them'. Adding to the buffer is an action that can easily cause issues if both threads try and change the values at the same time. The program uses a couple of methods to synchronize and manage common resources, such as waiting and notifying the updating common variables when one is operating on the buffer.
