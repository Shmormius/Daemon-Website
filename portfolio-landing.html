Autonomous Raspberry Pi Rover

C++

Python

html

This project was my submission for the CS 370 Operating Systems term project. The project rules were to group up into groups of 2-3 people and make something using a Raspberry Pi. In this process I learned a great deal about the development process and how to leverage hardware restrictions. We had the whole term to work so leveraging our time and avalibility to complete the project was another challenge.

We had decided on using our Pi to control a rover and developed a systems through which a user could provide coordinates over a desktop interface, and the rover would seek those coordniates. The program works by establishing a local server using flask on the Raspberry pi. Then the user would access the server via a webpage and type the coordniates. The Pi would then compute the instruction and send orders to an arduino that operates the rover. Not being robotics engineers the rover itself looks rather shabby but luckily functions. Using an onboard battery, a gps sensor, an arduino, and the PI itself its able to navagate to coordnates provided via the web.

Consumer Producer Problem

Java

This is my attempt at the producer-consumer problem using bounded buffer. Main starts two threads, producerThread and consumerThread which start the consumer and producer process. producer creates random double 'item' and checks if buffer size is equal to 1000, if it is, it waits for the consumer to consume one element. It adds the item to the buffer and icrements the counter of total produced items and adds the value to the sum of the produced items. consumer removes the oldest value in the buffer, maintains a value of the sum of the consumed items. Finally after enoughed items are consumed main takes the two threads and joins them back together.
